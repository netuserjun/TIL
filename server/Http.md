# Http
HyperText Transfer Protocol

http는 request와 response로 나뉜다.
굉장히 추상적인 개념인데
웹브라우저와 웹서버가 주고받는거다

웹브라우저는 사용자가 요청한 정보를 Request Headers를 만들어서 웹서버에게 대신 물어본다
